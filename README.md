# Task-Management-System
##Introduction

The Task Management System is designed to help Admins efficiently create, assign, and track tasks within a team. This system provides a minimal user interface where Admins can manage tasks, assign responsibilities, and monitor progress, enhancing overall productivity and task organization.

Features

1. Admin Registration and Authentication

Secure registration and login for Admins.

Authentication ensures controlled access to task management functionalities.

2. Task Creation and Assignment

Admins can create tasks with relevant details:

Title

Description

Deadline

Priority Level

Status

Tasks can be assigned to specific team members via email notifications.

3. Task Status Management

Track tasks efficiently with status updates:

Pending: Task has not been assigned yet.

In Progress: Task is assigned and currently being worked on.

Completed: Task is finished, and the Admin is notified.

4. Task Filtering and Querying

Search and filter tasks based on:

Task status

Due date

Assigned member

5. Task Prioritization

Admins can set priority levels:

High

Medium

Low

Sorting and filtering options to focus on important tasks.

6. UI Design and Enhancements

A simple and user-friendly interface for effective task management.

UI improvements for better usability and task visualization.

Deliverables

Schema Design: A normalized database schema for users, tasks, statuses, and deadlines.

SQL Queries: Queries for filtering tasks by status, due date, and assigned members.

Basic UI Implementation: Simple interface for task management.

Enhanced Features: Task prioritization, reporting, and UI improvements.

Installation

Clone the repository:

git clone https://github.com/your-username/task-management-system.git

Navigate to the project directory:

cd task-management-system

Set up the database using the provided SQL schema.

Configure authentication settings as needed.

Run the application and start managing tasks efficiently.
